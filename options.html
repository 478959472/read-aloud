Read Aloud Options

Voice
Max Length of Speechlet
(words)
If reading sometimes ends abruptly in the middle of a long sentence, try reducing this number. This setting causes longer paragraphs to be broken up into smaller speechlets.
Min Length of Last Paragraph
(words)
Instructs Chrome to stop reading when there are no more paragraphs that meet the specified length. If a lot of irrelevant stuff is read at the end of the article, try increasing this number. If the end of the article is cut off, try reducing it.
